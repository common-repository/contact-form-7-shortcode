Enable shortcodes for Contact Form 7

== Description ==

This plugin just enable shortcodes in forms created with Contact Form 7. 

Requires Contact Form 7 plugin to be installed.

== Installation ==

1. Upload the entire 'contact-form-7-shortcode' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

This plugin don't have a menu, or settings.
